## Featured work

### [NexusAI](https://github.com/Jacobdrosol/NexusAI)

NexusAI is a modular, distributed LLM control plane for coordinating specialized AI agents across local machines, GPUs, cloud models, tools, repositories, and private knowledge sources. It includes task orchestration, dependency-aware workflows, worker discovery and scheduling, model routing, project workspaces, knowledge retrieval, audit controls, observability, and an operator dashboard.

I am building it around a practical question: how can people combine specialized models and tools into useful workflows without hiding control, security, or failure handling behind a single opaque assistant? The platform is actively evolving, and the repository includes its architecture, operational documentation, security model, tests, and deployment tooling—not just a proof-of-concept interface.

### [Nexus Worker Node](https://github.com/Jacobdrosol/nexus-worker-node)

The worker node extends NexusAI across additional machines and compute backends. It is designed to make distributed model capacity easier to register, monitor, and use while keeping execution policies and deployment concerns explicit.

### [PersonalAssistant](https://github.com/Jacobdrosol/PersonalAssistant)

PersonalAssistant grew from the need to keep tasks, standups, deadlines, workflow context, and production responsibilities from becoming disconnected. It reflects the kind of software I enjoy building: tools grounded in an actual operational problem rather than features invented in isolation.
