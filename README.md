# Hi, I'm Jacob Derifield

I'm a full-stack software engineer who likes owning the whole problem: the interface, the API, the database, the infrastructure, and what happens after the software reaches production. **C# and .NET are my home base**, but I care more about building useful, dependable systems than collecting technologies.

At **CDS Global**, I support enterprise subscription-management products and help modernize workflows that span COBOL/mainframe systems and newer .NET and SQL applications. That means working with APIs and webhooks, validating production data, building middleware and automation, investigating failures, documenting how systems behave, and collaborating with the people who depend on the software every day.

Outside of work, I build products through **Derifield Enterprises LLC**. Some of that work stays private because it includes client sites, startup systems, and proprietary tools, but this profile is where I share the projects, experiments, and open-source work that I can make public.

## How I approach software

I enjoy taking complicated systems apart, understanding why they work, and rebuilding them so they are clearer and easier to operate. Production experience has taught me that good software is not just code that runs locally—it must handle real data, integrate with other systems, support real users, fail predictably, and remain understandable to the next person who maintains it.

My strongest tools are **C#, .NET, ASP.NET Core, Blazor, SQL, and PostgreSQL**. I also build with **TypeScript, JavaScript, React, Python, Java, and C++**, and work with **Linux, Docker, Proxmox, CI/CD, Azure, GCP, and AWS** when the problem extends into infrastructure and deployment.

Beyond application development, I'm interested in physics, scientific computing, electronics, IoT, PCB design, 3D CAD, game-engine tooling, and the point where software begins interacting with the physical world.
